Итак, вы решили выучить HTML?

Давайте разберемся в базовых терминах

  • HTML - язык гипертекстовой разметки. Мы используем его для размещения каких то данных на странице
  • Веб-страница - документ, который может быть отображён веб-браузерами, такими как: Firefox, Google Chrome, Microsoft Internet Explorer / Edge или Safari от Apple.
  • Веб-сайт - коллекция веб-страниц, связанных между собой какими-либо способами.

Итак, давайте попробуем создать свою "страницу"

Первое - что нам нужно сделать, это установить редактор кода.

  • Редактор кода - это текстовый редактор для создания и редактирования исходного кода программ. С его помощью мы можем удобно и быстро писать и редактировать код.

Моим выбором среди множества редакторов кода стал Visual Studio Code

После установки редактора кода

Создайте у себя на компьютере папку, можете назвать ее "Первый урок" и внутри этой папки создайте файл "index.html"

В файлах с расширением html мы будем писать наш HTML код.

Откройте ваш редактор кода, после чего нажмате File -> Open Folder -> выберите папку в которой вы создали index.html файл и откройте её

    Давайте произветем небольшую настройку нашего файла

  • Для начала я бы посоветовал в выбранном файле поставить восклицательный знак и нажать enter
  • У вас должен развернуться вот такой код
  • Внутри тега "body" напишите "Привет" или любой другой текст
  • С помощью сочетания клавиш ctrl + F5 вам откроется окно выбора браузера, и в выбранном браузере появится ваша первая Веб-страница.
